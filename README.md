# revealOnScroll #

jQuery Method for reveal objects onScroll with animation

##Usage##

###wrap the element on a div with a class 'rl-fadeObject' on the target object###

```
<div class="myelement rl-fadeObject">
   <h1>My Title</h1>
</div>
```

###call the method###

```
fadeObject('selector', speed);
```
